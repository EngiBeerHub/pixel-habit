# Product Overview

Pixel Habit は、Pixela API を利用して日々の行動記録を可視化するモバイルアプリです。  
主目的は「入力のしやすさ」と「ヒートマップによる継続モチベーション」を両立することです。

## Core Capabilities

- 認証情報（username/token）を使った Pixela 連携
- Graph（習慣）一覧の可視化と作成・編集・削除
- 日次 Pixel 記録の追加・更新・削除
- Graph 詳細での期間切替（Month/Year）と軽量な統計表示
- 設定画面からのセッション管理とトークン更新

## Target Use Cases

- 毎日の習慣実行を短い操作で記録したい
- 記録の積み上がりをヒートマップで直感的に確認したい
- 詳細画面で過去の記録と基本統計を振り返りたい

## Value Proposition

- Pixela の概念を維持しつつ、Expo Router ベースのネイティブ体験で扱いやすく提供する
- Home からの Quick Add で、記録操作の摩擦を最小化する
- 画面ごとに「表示責務」と「操作責務」を分離し、拡張時も一貫した UX を保ちやすい
