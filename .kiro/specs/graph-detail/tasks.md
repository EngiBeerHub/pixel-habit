# Implementation Plan

- [x] 1. Graph Detail の表示モード基盤を Short/Full 仕様へ移行する
- [x] 1.1 表示モードと期間決定ロジックを `short/full` と `14週/53週` に再定義する
  - 既存の表示モード定義を Short/Full 中心へ置換し、表示範囲を週数ベースで一貫化する。
  - Short は Habits と同等の14週、Full は53週を返す期間決定ルールを固定する。
  - _Requirements: 2.2, 2.3_
- [x] 1.2 データ取得条件とキャッシュ分離を新モード定義に同期する
  - 表示モード変更時に期間条件とデータ取得条件が必ず連動するよう状態管理を調整する。
  - モード単位でキャッシュが混線しないよう取得キーを分離し、同一モード再選択時の不要な再変化を防ぐ。
  - _Requirements: 2.5, 3.2, 4.4_

- [x] 2. Graph Detail の主要 UI をワイヤーフレーム準拠で再構成する
- [x] 2.1 表示モード切替 UI を Short/Full の2択として提供する
  - 切替操作の入力手段を2択に統一し、選択中モードを視覚的に判別可能にする。
  - _Requirements: 2.1, 2.4_
- [x] 2.2 ヒートマップを表示モード連動で更新し、read-only として扱う
  - Short/Full 選択に応じて14週/53週の可視化範囲を描画する。
  - セルタップによる記録追加導線は提供しない状態を明示する。
  - _Requirements: 1.2, 2.2, 2.3, 2.6_
- [x] 2.3 KPI チップセクションを追加し、統計をアイコン付きで表示する
  - アイコン・値・ラベルで構成される4指標（記録日数、累計、平均、今日）を同一セクションで表示する。
  - 記録なしケースではプレースホルダを用いて誤解なく表示する。
  - _Requirements: 3.1, 3.2, 3.3, 3.4_

- [x] 3. 画面状態・エラー処理・遷移整合を維持したまま統合する
- [x] 3.1 読み込み中・失敗・再試行のフィードバックを保持する
  - データ取得中の状態表示、失敗メッセージ、再試行操作を一連の体験として維持する。
  - _Requirements: 4.1, 4.2, 4.3, 4.4_
- [x] 3.2 Graph 識別情報と遷移文脈の一貫性を維持する
  - 画面タイトル/識別情報を表示し、戻る操作と再訪時表示ルールの一貫性を保つ。
  - 複数導線から遷移しても同等の主要情報を表示できる状態を維持する。
  - _Requirements: 1.1, 1.3, 5.1, 5.2, 5.3_

- [x] 4. 受け入れ条件に対応する自動テストを拡張する
- [x] 4.1 (P) 期間算出と統計整形の単体テストを更新する
  - Short/Full の範囲決定と記録なし時プレースホルダ表示を単体で検証する。
  - _Requirements: 2.2, 2.3, 3.3, 3.4_
- [x] 4.2 Graph Detail の統合テストを Short/Full 挙動に合わせて更新する
  - 初期表示、モード切替、同一モード再選択時の挙動、取得条件同期を検証する。
  - _Requirements: 2.1, 2.4, 2.5, 3.2, 4.4_
- [x] 4.3 KPI 表示と read-only 制約の UI テストを追加する
  - KPI 4指標のアイコン・値・ラベル表示、およびセルタップ追加導線不在を検証する。
  - _Requirements: 2.6, 3.1, 3.2_
- [x] 4.4 エラー・再試行・遷移の回帰テストを更新する
  - 取得失敗時の表示、再試行復帰、戻る導線と再訪整合を検証する。
  - _Requirements: 4.1, 4.2, 4.3, 4.4, 5.1, 5.2, 5.3_

- [x] 5. 実装統合と品質ゲート通過を完了する
- [x] 5.1 実装差分を結合し、表示・取得・遷移を統合確認する
  - 主要フロー（初期表示、Short/Full切替、KPI同期、履歴導線）が連続して成立することを確認する。
  - _Requirements: 1.2, 2.1, 2.2, 2.3, 3.2, 5.3_
- [x] 5.2 品質ゲート（lint/test）を通過させ、回帰リスクを解消する
  - 既存機能を含む回帰がないことを確認し、失敗時は原因を修正して再実行する。
  - _Requirements: 1.1, 1.3, 4.1, 4.2, 4.4, 5.1_
