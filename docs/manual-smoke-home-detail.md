# Manual Smoke: Home + Detail (5分)

この手順は、Home最小導線とDetail管理導線の回帰確認を短時間で行うためのチェックリストです。

## 事前条件

- 認証済みユーザーでログイン済み
- Homeに最低1件以上のグラフが表示される

## チェック手順

1. Homeで導線を確認する
- グラフカード内に `記録一覧` / `統計` / `編集` / `削除` / `...` が表示されないこと
- カード上の `+` が表示されること
- ヒートマップセルがタップできること

2. HomeからQuick Addを開く
- カード `+` を押してBottom Sheetを開く
- 日付初期値が今日（`yyyyMMdd`）であること
- 任意メモ欄（複数行入力可）が表示されること
- `保存`（primary）のみが表示されること（`詳細入力へ` は存在しないこと）

3. Quick Add保存とToastを確認する
- 数量を入力して `保存`
- 任意メモ（例: `朝ラン`）を入力した場合も保存できること
- シートが閉じること
- HeroUI Nativeのsuccess toast（デフォルト見た目）が表示されること
- 保存成功時に主要操作の触覚フィードバックが発生すること

4. Detail遷移と期間切替を確認する
- カード本体タップで `/graphs/[graphId]` に遷移すること
- `Month` / `Year` 切替が動作すること
- `Month=暦月 / Year=暦年` の補助文言が表示されること

5. 記録行から記録詳細へ遷移する
- Detailの記録一覧行をタップ
- `/graphs/[graphId]/pixels/[date]` へ遷移すること
- 記録一覧でメモあり行のみ要約が1行表示されること（改行は空白に正規化）
- 記録詳細でメモ（複数行）を更新できること
- 画面で更新または削除を実行し、完了後に戻れること

6. スクロールバー表示を確認する
- Home / Graph Detail / Graph Create / Graph Edit / Settings で、スクロールバーが常時表示されないこと
- iOS/Androidで同じ確認を行い、プラットフォーム差分がないこと

7. Settings導線を確認する
- Settingsトップから `トークン変更` 画面に遷移できること
- トークン変更画面で更新成功時のメッセージ表示が行われること

## 合格基準

- Home導線が `セル/+/カード` のみである
- Quick Add成功時の通知がToastで統一されている
- Detailで期間切替と記録詳細遷移が成立する
- optionalData（メモ）が保存・要約表示・更新まで一貫して成立する
- 記録更新/削除後に導線が破綻しない
- 通常操作中にスクロールバーが常時表示されない
